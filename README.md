# x86 Bare Metal Bootloader
16 -> 32 -> 64 Bit mode in assembly

## WHAT IS THIS?
A SIMPLE BOOTLOADER MADE IN ASSEMBLY

## NEED
- NASM (assembler)
- QEMU (emulator)

## BUILD
```
make
```

## RUN
```
make run
```

## CLEAN
```
make clean
```

## WORKS?
- SIMPLE BOOTLOADER
- PRINTING 'A' IN 16 BIT MODE
- JUMP TO 32 BIT MODE
- JUMP TO 64 BIT MODE
- PRINT 'k' IN 64 BIT MODE
- END PROGRAM
